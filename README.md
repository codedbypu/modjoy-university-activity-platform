# ProjectWeb
## Concept

เว็บไซต์/แอปพลิเคชันที่ทำหน้าที่เป็น **แพลตฟอร์มกลาง** สำหรับ

- เชื่อมต่อคนใน มจธ ที่ต้องการทำกิจกรรมเดียวกัน

## Target

- คนที่ต้องการหาเพื่อนช่วยกันติวในรายวิชาเดียวกัน
- คนที่เหงา อยากหาเพื่อนทำกิจกรรมด้วย
- คนที่มีความสนใจเรื่องเดียวกัน
- คนที่อยากหาทีมไปแข่งตามรายกรต่างๆ

## Features

1.สร้างบัญชีผู้ใช้ (เมลมอ และหน้าของผู้ใช้)

2.สร้าง/ลบ/แก้ไข ห้องที่สามารถจอยเข้ามารวมตัวกันทำกิจกรรทต่างๆร่วมกันได้

3.ระบบคะแนนความน่าเชื่อถือของ User

4.ห้องแยกเป็นหมวดหมู่ เช่น ห้องติว, ห้องนัดกิจกรรม, ห้องนัดกินข้าว, ห้องนัดเล่นเกม

5.ระบบการเช็คชื่อภายในห้อง

**6.ระบบ Matching/Recommendation**

- เช่น แนะนำเพื่อน/กิจกรรมที่น่าจะเข้ากับผู้ใช้ จาก ** tag ความสนใจ **

## ด้าน Database & CRUD

- ตารางหลัก: Users, Rooms, RoomTag, UserTag, Roommember, Locations, faculty, Tags
- Feature CRUD:
    - User: สมัคร/แก้ไขโปรไฟล์/ลด-เพิ่มคะแนน
    - Room: สร้าง/แก้ไข/เข้าร่วม/ลบ
    - Roommember: Join/ยกเลิก/เช็คชื่อ
    - Tags: สร้างอัตโนมัติ
    - RoomTag/UserTag: สร้างอัตโนมัติ

## เงื่อนไขระบบเช็คชื่อ
- ไม่สามารถเปิดได้ถ้าเวลาปิดกิจกรรม - เวลาปัจจุบัน แล้วน้อยกว่าหรือเท่ากับ 10นาที
- ถ้าสร้างห้องต่ำกว่า 15 นาที ให้ปิดฟังก์ชันเช็คชื่อไปเลย
- ต้องอยู่ในเวลากิจกรรมถึงจะเปิดการเช็คชื่อได้

## ขอบเขตของงาน
- ไม่รองรับกิจกรรมที่ใช้เวลาเกิน 1วัน
- ไม่รองรับระบบ chat
- ไม่รองรับการยืนยันผ่านมหาวิทยาลัย
- ไม่รองรับการแจ้งเตือน
- ไม่รองรับระบบรายงานความเหมาะสมห้อง
- ไม่รองรับระบบการดึงข้อมูลข่าวสาร Official

## ต้องลงอันนี้ก่อน
1. ลง Database ใน MySQL Workbench(MySQL) ใช้ไฟล์ `inint.sql` ในโฟล์เดอร์ database 
2. ลง npm และ extension : `npm install jsonwebtoken cookie-parser multer node-cron`
3. ลงโปรแกรม XAMPP ก่อน
4. ก็อป .env.example มาอีกไฟล์แล้วเปลี่ยนนามสกุลเป็น .env
5. ตั้งค่าตัวแปร และ Port ที่จะใช้ใน .env

## วิธี Run
1. Run `server.js` ก่อน
2. เปิด Browser แล้วเข้าตาม `http://localhost:<PORT>/` ได้เลย
