# ProjectWeb
## Concept

เว็บไซต์/แอปพลิเคชันที่ทำหน้าที่เป็น **แพลตฟอร์มกลาง** สำหรับ

- เชื่อมต่อคนใน มจธ ที่ต้องการทำกิจกรรมเดียวกัน

## Target

- คนที่ต้องการหาเพื่อนช่วยกันติวในรายวิชาเดียวกัน
- คนที่เหงา อยากหาเพื่อนทำกิจกรรมด้วย
- คนที่มีความสนใจเรื่องเดียวกัน

## Features

1.สร้างบัญชีผู้ใช้ โดยที่ต้องมีการยืนยันตัวตน (เมลมอ และหน้าของผู้ใช้)

2.สร้างห้องที่สามารถจอยเข้ามารวมตัวกันทำกิจกรรทต่างๆร่วมกันได้

3.ระบบคะแนนความน่าเชื่อถือของ User

4.ห้องแยกเป็นหมวดหมู่ เช่น ห้องติว, ห้องนัดกิจกรรม, ห้องนัดกินข้าว, ห้องนัดเล่นเกม

5.เขียนบทความลงหน้าเว็บได้

**6.ระบบ Matching/Recommendation**

- เช่น แนะนำเพื่อน/กลุ่มที่น่าจะเข้ากับผู้ใช้ จาก **วิชาที่เรียนเหมือนกัน / tag ความสนใจ / เวลาเรียนว่างตรงกัน**
- ทำให้ดูไม่ใช่แค่ CRUD แต่มีการใช้ **query ที่ซับซ้อน + algorithm เล็กๆ**

7.**ระบบ Notification**

- แจ้งเตือนถ้ามีคนกด join ห้องเรา, มีคอมเมนต์ใหม่ หรือ event ที่ใกล้จะถึง
- อาจเริ่มจาก **notification บนเว็บ** (ไม่ต้องถึง push จริงๆ) ก็เพิ่มความสมจริงได้เยอะ

8.**Gamification (Badge/Level)**

- เช่น คนที่เข้ากิจกรรมบ่อย ได้ badge "Active Member"
- หรือถ้ามีคนให้ rating ดี → ได้ “Trusted Member”
- จะทำให้ระบบน่าเล่นและดู advanced

**9.ระบบ Chat/Message ในห้อง**

- **แบบ real-time (ใช้ WebSocket / Firebase) หรือแบบง่ายคือข้อความในฐานข้อมูลแล้ว refresh**
    - **ทำให้เว็บดู interactive และท้าทายกว่าการเก็บข้อมูลเฉยๆ**

## ด้าน Database & CRUD

- ตารางหลัก: Users, Rooms, RoomCategories, Posts/Articles, Events, Messages, Ratings
- Feature CRUD:
    - User: สมัคร/แก้ไขโปรไฟล์/ลบ
    - Room: สร้าง/แก้ไข/เข้าร่วม/ลบ
    - Post/Article: เขียน/แก้ไข/ลบ
    - Event: สร้าง/Join/ยกเลิก
    - Rating: ให้คะแนน/คอมเมนต์

## UX/UI ที่เพิ่มคะแนนได้

- **Search + Filter**: เช่น ค้นหาห้องด้วยคีย์เวิร์ด, filter ตามหมวดหมู่, วันเวลา
- **Profile Page**: เห็นกิจกรรมที่เคย join, คะแนนความน่าเชื่อถือ, ความสนใจ
- **Dashboard**: สำหรับดูว่ามีกิจกรรม/ห้องที่ join ไว้วันไหนบ้าง

## ต้องลงอันนี้ก่อน
1. ลง Database ใน MySQL Workbench(MySQL) ใช้ไฟล์ `inint.sql` ในโฟล์เดอร์ database 
2. ลง npm และ extension : `npm install jsonwebtoken cookie-parser multer node-cron`
3. ลงโปรแกรม XAMPP ก่อน
4. ก็อป .env.example มาอีกไฟล์แล้วเปลี่ยนนามสกุลเป็น .env
5. ตั้งค่าตัวแปร และ Port ที่จะใช้ใน .env

## วิธี Run
1. Run `server.js` ก่อน
2. เปิด Browser แล้วเข้าตาม `http://localhost:<PORT>/` ได้เลย
